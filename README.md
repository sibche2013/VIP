# SNI IP Scanner v3.1

A powerful, real-time SNI (Server Name Indication) IP scanner with a sleek glassmorphism UI. Scans background IP pools to find clean, working IPs for premium SNI targets and generates Trojan protocol configurations.

---

## Features

- Real-time IP scanning from a pool of 110+ domains
- DNS resolution via multiple providers (Cloudflare, Google, Alibaba, Tencent)
- Automatic ping measurement and response time tracking
- Trojan URI config generator with SNI profile selection
- Light/Dark theme toggle
- Copy-to-clipboard config export
- Fully responsive design (mobile + desktop)

## Demo

Open `index.html` in any modern browser. No server required.

## How It Works

1. **Background IP Pool** — A curated list of 110+ domains is resolved and pinged in parallel batches.
2. **SNI Target Matching** — Working IPs are mapped to 5 premium SNI targets.
3. **Config Generation** — Click any working row, select an SNI profile, and get a ready-to-use Trojan config URI.

## Tech Stack

- HTML5 / CSS3 (Glassmorphism + Tailwind CSS)
- Vanilla JavaScript (no framework, no build step)
- Google Fonts (Inter + JetBrains Mono)
- Cloudflare DNS-over-HTTPS API
